Video tutorials
===============

The *TVB EduPack* on the **TVB main website** contains `dozens of video lectures <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase>`_ explaining TVB's GUI and scripting interfaces. Some lectures help you reproduce the research done for a peer-reviewed neuroscience publication:

* *Basics:* `My first simulation with The Virtual Brain <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42186-learn-my-first-simulation-with-the-virtual-brain>`_, lecture by Paul Triebkorn
* *Basics:* `Population Models in The Virtual Brain (TVB) and the Phase Plane <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42198-learn-population-models-in-the-virtual-brain-tvb-and-the-phase-plane>`_, lecture by Dr. Michael Schirner
* *Epilepsy:* `Modelling Epilepsy with The Virtual Brain <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42218-learn-modelling-epilepsy-with-the-virtual-brain>`_, lecture by Julie Courtiol
* *Epilepsy:* `The Bayesian Virtual Epileptic Patient (BVEP) <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42230-learn-the-bayesian-virtual-epileptic-patient-bvep->`_, lecture by Meysam Hashemi
* *Stimulation:* `Brain stimulation in The Virtual Brain <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42233-learn-brain-stimulation-in-the-virtual-brain>`_, lecture by Andreas Spiegler
* *Stimulation:* `Surface-based simulations and deep brain stimulations <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42240-learn-surface-based-simulations-and-deep-brain-stimulations>`_, lecture by Jil Meier
* *Stroke:* `TVB Clinical Applications - Stroke Recovery & Dementia <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42247-learn-tvb-clinical-applications-stroke-recovery-dementia>`_, lecture by Randy McIntosh
* *Stroke:* `Hands-On: Modeling stroke brain <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42250-learn-hands-on-modeling-stroke-brain>`_, lecture by Paul Triebkorn
* *Model construction:* `Import Virtual Brain ready data into TVB and create a brain model <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42260-learn-import-virtual-brain-ready-data-into-tvb-and-create-a-brain-model>`_, lecture by Patrik Bey
* *Model construction:* `Integrated workflows: Image processing pipeline <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42266-learn-integrated-workflows-image-processing-pipeline>`_, lecture by Dr. Michael Schirner
* *Special applications:* `Modeling brain dynamics in brain tumor patients using The Virtual Brain <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42279-learn-modeling-brain-dynamics-in-brain-tumor-patients-using-the-virtual-brain>`_
* *Special applications:* `Simulating The Virtual Mouse Brain (TVMB) <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42282-learn-simulating-the-virtual-mouse-brain-tvmb->`_, lecture by Patrik Bey
* *Advanced simulation:* `Human Brain Project (HBP) TVB-NEST co-simulation <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42297-learn-human-brain-project-hbp-tvb-nest-co-simulation>`_
* *Advanced simulation:* `TVB to NEST multi-scale simulation <https://www.thevirtualbrain.org/tvb/zwei/newswire-educase/single/42313-learn-tvb-to-nest-multi-scale-simulation>`_, lecture by Dionysios Perdikis
